DATA STRUCTURE:
All data is stored per user:
- users/{userId}/stores/{storeId}/inventory

- users/{userId}/settings/app

This means:
- Same user logs in on web or mobile → sees same data
- Data syncs automatically between platforms
- Each user's data is isolated
